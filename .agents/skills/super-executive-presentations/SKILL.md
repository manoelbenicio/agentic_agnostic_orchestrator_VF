# 🎨 Super Executive Presentations — Fortune 500 Storytelling & PPTX

## Overview

Custom-built elite skills for creating Fortune 500-grade executive presentations. These skills work together as a pipeline: narrative architecture → content generation → visual KPI design → animation.

## When to Use This Skill Group

- Creating executive status reports for C-Level audiences
- Building presentation narratives with 7-act story arcs
- Generating McKinsey/BCG/Bain-tier content
- Designing premium PPTX KPI components (gauges, progress bars, hero KPIs)
- Adding professional PowerPoint animations via python-pptx
- Product launch presentations in Steve Jobs style

```
1. ppt-narrative-architect  →  Defines the ROTEIRO (story structure)
```

## Overview

Best-in-class skills for designing, building, and optimizing multi-agent systems. From LangGraph state machines to Kanban-based autonomous pipelines.

- Implementing supervisor/planner/worker agent patterns

## Key Patterns

1. **Pipeline Sequential** — output of one agent feeds the next
2. **Parallel Execution** — independent agents work simultaneously
3. **Primary + Support** — one lead agent with data-support agents
4. **Deterministic Before Probabilistic** — hard gates before AI review
5. **Adversarial Review** — different agent reviews code critically

# 🤖 Super AI Agents — Agent Architecture & Development

## Overview

Elite collection of the best AI agent skills for designing, building, testing, and deploying autonomous AI agents. These skills cover the full lifecycle from architecture to production monitoring.

## When to Use This Skill Group

- Building autonomous AI agents (single or multi-step)
- Designing agent memory architectures (short-term, long-term, episodic)
- Creating agent tools (MCP servers, function calling, JSON Schema)
- Evaluating and benchmarking agent performance
- Building agents on Azure AI Foundry or M365

## Key Principles (from top skills)

1. **Reliability over autonomy** — every step compounds error probability
2. **Constrain scope** — domain-specific beats general-purpose
3. **Memory quality = retrieval quality**, not storage quantity
4. **Tool descriptions > implementations** — LLMs only see schemas
5. **Test tools with the LLM**, not just unit tests
6. **Treat outputs as proposals**, not truth

HerdMaster Micro Architecture

Component internals mapped to committed modules. Each panel shows the active runtime path and background asyncio tasks that keep the control plane moving.

Message Bus Internals

bus.messages defines typed JSON-RPC messages; bus.server hosts the asyncio Unix-socket broker with queues, persistence, TTL sweep, and file fallback.

Queue + Dispatch Internals

TaskQueue orders ready work and claims with SQLite CAS; DispatchInjector resolves panes, waits idle, sends chunks, or writes prompt files for long payloads.

Watchdog + Recovery Internals

WatchdogEngine combines primary events, secondary polling, and tertiary output hashes. RecoveryManager interrupts, respawns, replays, and escalates.

Project Mode Internals

ProjectPlanner, SquadRecommender, and EtaEstimator turn one scope into a reviewed project plan and concrete queue tasks.

ACL Engine Internals

Configuration is parsed into roles, wildcard mappings, target-role resolution, and enforcement checks for Control API message routes and bus sends.

Control API + CLI Internals

herdmaster.cli is the composition root; ControlApiServer is the local API router over Unix socket and optional token-protected localhost HTTP.

Asyncio Task Relationships

The foreground CLI process launches cooperating services on one event loop. Durable state sits in SQLite while Herdr integration remains behind the adapter boundary.

Bus task Unix server accepts clients; subscriber pumps run per writer.
Watchdog task main loop plus optional primary listener; polling continues if primary is absent.
Dispatch task scans idle/healthy agents, CAS-claims work, then injects prompts.
API task one route dispatcher covers system, tasks, projects, agents, messages, metrics.
Storage SQLite stores durable state; config is TOML with hot reload support.
